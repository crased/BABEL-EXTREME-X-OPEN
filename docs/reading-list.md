# Reading List

A flat list of tools and papers you may find useful. **This is not a
recommendation.** Some of these are excellent for this task. Some are
mediocre. Some are wrong for the job in ways that aren't obvious. Some have
been superseded. Working out which is which is part of the assignment.

We have not sorted, ranked, or annotated. Read them or don't.
